# 浮光绘匣 · Prompt Atelier

本地优先的画师串、角色词、服装和场景提示词收藏与组合工具。

## 当前版本

- 角色、画师/风格、服装、表情/动作、场景/镜头五类模块收藏
- 模块多选与批量全组合
- 组合方案保存与载入
- 单项复制、TXT/JSON 批量导出
- 全资料 JSON 备份
- 普通 PNG `tEXt`、`zTXt`、`iTXt` 元数据读取
- PWA 骨架与手机端响应式界面

资料默认保存在浏览器的本地存储中，不会上传到网络。清除浏览器站点数据前，请先点击“备份资料”。

## 启动

Windows 可以直接双击 `启动串匣.cmd`。本地服务确认启动成功后，网页会自动打开；保持黑色服务窗口开启，关闭它即可停止串匣。

也可以在项目目录手动运行（电脑需要安装 Node.js）：

```powershell
npm start
```

然后打开：<http://127.0.0.1:4173>

关闭时在运行窗口按 `Ctrl+C`。

## 测试
